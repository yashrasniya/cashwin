# Cashwin

Create a web application allowing users to engage in games such as Card32 and Teen Patti, AUG 2022- DEC-2022 where they earn rewards following each victorious round. The backend is constructed using Django, while the frontend is built with React. API functionality is provided through REST-API and Socket.IO, with JWT employed for authentication. All data is stored in an SQL server, and the application is hosted on an AWS server.
